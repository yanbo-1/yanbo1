# 机械零件同心度检测系统

基于机器视觉的圆柱形机械零件（轴套）同心度自动化检测系统，采用 Python + OpenCV + PyQt5 开发，支持图像采集、预处理、圆心检测、同心度计算、结果可视化与报告导出。

## 🎯 功能特性

- ✅ 图像采集支持（摄像头实时采集/本地图像导入）
- ✅ 图像预处理流程（反光抑制、噪声去除、边缘优化）
- ✅ 双圆心检测算法（改进霍夫变换 + 最小二乘拟合）
- ✅ 透视畸变矫正（支持相机标定与在线矫正）
- ✅ 同心度计算与偏差分析（支持单位换算与容差判断）
- ✅ 可视化界面（图像实时显示、圆心标记、偏差标注、曲线绘制）
- ✅ 参数动态调节与保存
- ✅ 检测报告导出（Excel/PDF）
- ✅ 一键检测、历史数据查询、批量检测支持
