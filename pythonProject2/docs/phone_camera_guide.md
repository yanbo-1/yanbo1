# 手机摄像头连接使用指南

## 1. 准备工作

### 1.1 安装手机应用
在手机上安装IP摄像头应用，推荐：
- **安卓**：IP Webcam
- **苹果**：IP Camera

### 1.2 网络设置
确保手机和电脑连接到同一个WiFi网络

## 2. 连接步骤

### 方法一：通过文件加载
1. 使用手机拍摄零件图像
2. 将图像传输到电脑
3. 在软件中选择"从文件加载"

### 方法二：实时摄像头连接
1. 在手机上打开IP摄像头应用
2. 启动服务器（通常点击"启动服务器"）
3. 记下显示的IP地址和端口（如：192.168.1.100:8080）
4. 在软件中选择"连接手机摄像头"
5. 输入摄像头URL（如：http://192.168.1.100:8080/video）

## 3. 拍摄建议

### 最佳实践：
1. 使用后置摄像头（分辨率更高）
2. 关闭美颜滤镜
3. 开启HDR模式
4. 固定焦距
5. 垂直拍摄零件
6. 确保光线充足

### 拍摄参数：
- 分辨率：1920×1080或更高
- 距离：约30-50cm
- 背景：纯色背景（白色或黑色）
- 角度：正对零件中心

## 4. 故障排除

### 无法连接：
1. 检查WiFi连接
2. 检查防火墙设置
3. 重启手机应用
4. 重新连接WiFi

### 图像模糊：
1. 清洁镜头
2. 调整焦距
3. 增加光照
